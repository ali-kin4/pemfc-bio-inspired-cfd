Features Implemented:
--------------------
✓ Tapered interdigitated cathode flow field (15% area reduction)
✓ Two-level lung-inspired inlet manifold (area-preserving branching)
✓ Multi-physics coupling (flow, heat, species, electrochemistry)
✓ Brinkman-Forchheimer porous media modeling
✓ Butler-Volmer electrochemistry with temperature dependence
✓ Three mesh refinement levels (coarse/medium/fine)
✓ Mesh independence study framework
✓ Operating matrix study (3 current densities × 2 stoichiometries)
✓ Publication-quality output generation
✓ Complete reproducibility framework

Usage Examples:
--------------
# Basic test
python test_basic.py

# Demo run
python notebooks/demo.py

# Single case simulation
python src/main.py --mesh medium --current 1.2 --stoich 2.0

# Mesh independence study
python src/main.py --study mesh_independence

# Operating matrix study
python src/main.py --study operating_matrix
